Improve the model

          
Each saved correction makes the next training run more accurate on your vendor formats.

          
    
            
  1. 
              Collect corrections
              Save fixes in Step 4 → data/review/corrections.csv
            
    2. 
            
  2. 
              Regenerate dataset
              python tools/generate_training_dataset.py --count 2000 --merge-corrections
            
    3. 
            
  3. 
              Fine-tune
              python tools/finetune_unsloth.py --data data/training/train.jsonl --epochs 2
            
    4. 
            
  4. 
              Deploy to Ollama
              ollama create arthavi-address -f artifacts/arthavi-address/Modelfile
            
    5. 
            
  5. 
              Evaluate accuracy
              python tools/evaluate_mapper.py --use-llm --model arthavi-address
            
    6. 
          

          
Target: ≥95% field accuracy on held-out addresses after retrain.
